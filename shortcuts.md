## Macos shortcuts

| Shortcut | Application | Comment                             |
| -------- | ----------- | ----------------------------------- |
| ⌘[       | Finder      | Moves back to the previous location |
| ⇧⌘c      | iTerm2      | Copy mode to use Vim-like shortcuts |
| ⌘⌫       | iTerm2      |  Delete back to start               |

## Firefox and Vimium shortcut

| Shortcut | Application | Comment           |
| -------- | ----------- | ----------------- |
| ⌘⌃←      |  Firefox    |  Jump to left tab |
